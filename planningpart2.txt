### ⚙️ Updated Transpiler Logic (`transpiler.py`)
Conversational syntax needs slightly smarter pattern matching. Here’s a clean, scalable way to handle it:

```python
import re

def transpile(code: str) -> str:
    lines = code.split("\n")
    new_lines = []

    for line in lines:
        stripped = line.strip()
        
        # 1. Conversational IF condition
        # Matches: jodi <var> <num>t koi besi hoi, tetia:
        match_if = re.match(r"jodi\s+(.+?)\s+(\d+)t\s+koi\s+besi\s+hoi,\s*tetia:", stripped)
        if match_if:
            left = match_if.group(1).strip()
            right = match_if.group(2)
            new_lines.append(f"if {left} > {right}:")
            continue

        # 2. Conversational ELSE
        if stripped == "nohole ba:":
            new_lines.append("else:")
            continue

        # 3. Print (kua)
        if stripped.startswith("kua "):
            content = stripped[4:].strip()
            new_lines.append(f"print({content})")
            continue

        # 4. Variable declaration (dhora)
        if stripped.startswith("dhora "):
            new_lines.append(stripped[6:])  # removes "dhora "
            continue

        # 5. Fallback: keep line as-is (Python-compatible lines)
        new_lines.append(line)

    return "\n".join(new_lines)
```

```python
import sys
from ooju.transpiler import transpile

def main():
    if len(sys.argv) < 2:
        print("Usage: ooju run <file.oj>")
        return

    file_path = sys.argv[1]
    try:
        with open(file_path, "r", encoding="utf-8") as f:
            code = f.read()
        
        py_code = transpile(code)
        exec(py_code)
    except FileNotFoundError:
        print(f"❌ File '{file_path}' not found.")
    except Exception as e:
        print(f"🔍 Ooju Error: {e}")

if __name__ == "__main__":
    main()
```

### 🧠 Why This Conversational Syntax Works (And How to Keep It Stable)
✅ **Pros:**
- Reads like a WhatsApp message → zero syntax anxiety
- `tetia:` acts as a natural "then" marker
- `nohole ba:` feels like "otherwise, but..." (very Assamese)

⚠️ **Parsing Reality Check:**
Conversational = flexible for humans, but **rigid for machines**. To avoid future bugs:
1. Lock the pattern: `jodi <value> <number>t koi <operator>, tetia:`
2. Don’t allow free-form variations yet (e.g., `jodi 5t koi besi hoi coin, tetia:` will break simple regex)
3. Later, switch to a **parser (Lark/ANTLR)** when you add `kom hoi`, `soman`, or nested conditions
